================================================================================
CASE 1: Sequential Batching - Conversion Worker
================================================================================
Demonstrates: Page-based dynamic batching in the conversion worker

## Key Behavior:
- Documents are batched based on page count (limit: 100 pages)
- Once page limit is exceeded, batch is processed immediately
- Sequential document processing within each batch

## Test Scenario:
- 3 documents: 30, 40, 50 pages
- Expected: All 3 docs batched together (120 pages exceeds 100 limit)
